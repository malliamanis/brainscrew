# Brainscrew
Brainfuck but with words. Includes a Brainscrew interpreter, a Brainscrew to C compiler, a Brainscrew to Brainfuck compiler and a Brainfuck to Brainscrew compiler.

## How to build
for debug
```
$ make
```
for release
```
$ make release
```

---

## How to use
```
$ ./build/release/bfci -h
```
for help
